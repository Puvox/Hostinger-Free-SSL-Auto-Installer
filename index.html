Setup HTTPS for your site with Let's encrypt free SLL

NOTE: This script automatically emails you the generated CERTIFICATE and PRIVATE KEY (Remember to check spambox, sometimes it gets there). After that, Apply the keys to your hosting SSL page.

STEPS:

1) Download the .zip from GitHub , unpack and upload the contents of the folder in public_html (the folder name will be auto_ssl, however, I advise to rename to something unique for yourself)
2) Edit cron.php file (only your email & correct paths needed).
3) Setup a cron job to run the below code (once in 2 months):
wget -O /dev/null http://yoursite.com/auto_ssl/cron.php
4) Once in 2 month, you will get an EMAIL containing the CERT and PRIVATE key (Check SPAMBOX, mostly there).
(BTW, now, at least once, test that url in your browser to get the keys).
