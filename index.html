Setup HTTPS for your site with Let's encrypt free SLL

NOTE: This script automatically emails you the generated CERTIFICATE and PRIVATE KEY (Remember to check spambox, sometimes it gets there). After that, Apply the keys to your hosting SSL page.

STEPS:

1) Download the .zip from GitHub , unpack and upload the contents of the folder in PARENT DIRECTORY of public_html.
2) Edit cron.php file and set your email & correct paths.
3) Setup a cron job to run once in 1 minute (after first successful run, then QUICKLY change that to once in a 2 month, otherwise it will make many requests with 429 error), type the below code (just change to your credentials):
/usr/bin/php /home/uXXXXXXXXXXX/domains/yoursite.com/cron.php basename=yoursite.com&domains[]=yoursite.com&domains[]=www.yoursite.com
(Note: Remove www subdomain from parameters if you don't have that subdomain accessible.)
